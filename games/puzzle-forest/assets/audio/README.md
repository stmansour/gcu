Puzzle Forest voice generation sources live here.

Current game state:

- gameplay prompts are using browser `speechSynthesis`
- success and mistake feedback are using lightweight generated tones

These text files are the source material for replacing the browser voice with real narrated audio generated by `piper`.

Suggested workflow:

1. Generate one audio file per line entry.
2. Keep filenames stable so code can map to them later.
3. Prefer short, warm, upbeat delivery with clear diction for ages 2 to 4.
4. Export compact `.mp3` or `.wav` assets depending on the runtime path you choose.

Manifest format:

- `slug|spoken text`
- one entry per line
- slugs are lowercase and stable for later scripting

Recommended voice direction:

- warm adult guide
- slow but not sleepy
- encouraging, playful, emotionally safe
- no exaggerated announcer tone
